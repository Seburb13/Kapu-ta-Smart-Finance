Test Plan #TP001

Project: Kapu$ta Smart Finance         Site: Kapusta (kapusta-qa.netlify.app)     Owner: Sebastian Urbański

Release: Release April 2023              Sprint: 20 March - 03 April 2023

Introduction

Shows the basic approaches to test web and mobile application Kapu$ta Smart Finance. There is no additional info about this application included, except mockup files: Kapusta – Figma containing desktop and mobile print screens looks like wallet application and List of project  requairements.

Goals

1) Create test documentation: Test cases and scenarios, Test Plan, Test Report, additionally grouping into Test suite by choosen Selection Key.

2)  Create Bug Report with all incompatible results of tests conducted.

*3) QA3: Project approved by Mentor

Risks

1) Time Issue

2)  Issues with definition of right requirements

3) Personal data safety - spy software

*4) QA3: Certificate in GoIT without course finished annotation

List of project requairements

Kapusta (kapusta-qa.netlify.app)

Kapusta – Figma

Out of Scope

Load testing, HTTP URI

Browser: Microsoft Edge ver. 111.0.1661.44, 64-bit

Mockup: Figma

1) List of project  requairements and Mockup  are clear for testers

2) All test environments and test tools are available and can be used

3) Testers are able to meet the goals entrusted to them and use the right tools

People involved:
